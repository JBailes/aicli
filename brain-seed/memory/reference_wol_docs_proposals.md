---
name: wol-docs proposal structure
description: Where design proposals live and how they are organized
type: reference
---

All design proposals (except acktng-only bugfixes) live in `wol-docs/proposals/`:
- `proposals/pending/` — awaiting discussion or approval
- `proposals/active/` — currently being implemented
- `proposals/complete/` — fully implemented
- `proposals/rejected/` — rejected

acktng-specific proposals (bugfixes or changes with no effect outside acktng) remain in `acktng/docs/proposals/`.
