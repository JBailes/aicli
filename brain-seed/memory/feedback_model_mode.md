name: Model mode for proposals vs implementation
description: Use Opus for writing plans/proposals, Sonnet for implementation
type: feedback
---

Use **Opus** mode when writing plans and design proposals. Use **Sonnet** mode when implementing code.

**Why:** User preference — proposals benefit from deeper reasoning; implementation is faster in Sonnet.

**How to apply:** Switch to `/model opus` before drafting any design proposal or plan. Switch back to `/model sonnet` before writing code.
